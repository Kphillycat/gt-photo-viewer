# GT Photo Viewer

This is an application that allows the viewing of photos in a responsive gallery format from this [source](https://pastebin.com/BmA8B0tY). Features include:

- Pagination
- Filtering
- Grayscale toggling

![demo](./images/gt-photo-viewer.gif)
